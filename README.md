# Windows-10-System-Programming

Here I post my solutions to  tasks from the book
"Windows 10 System Programming" (by Pavel Yosifovich).

## Goal of the Project:
- Practical study of Windows system programming and Windows Template Library (WTL) for developing Windows applications and UI components. 
- creating a Portfolio with examples of working with system calls, Win API and WTL.

## Project Structure:
This project is split in chapters.
Each chapter has README section and 'src' folder with exercices.
**IMPORTANT: Pay attention to WTL section, where i explain how to install WTL and avoid errors that i encountered.

## Development environment:
Code was tested in Windows 10 and Visual Studio 2022.

## Contents by chapters:
- Chapter 1 describes basic Windows system information and how to get it using Windows API.
